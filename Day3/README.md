# Python Beginner Projects

This repository contains my beginner Python projects while learning Python.

## Projects

1. Age Calculator
- Takes user's age as input
- Categorizes age using conditional statements

2. Electricity Bill Calculator
- Calculates electricity bill based on units
- Uses if-elif-else conditions

3. Getting Time Greeting
- Takes time as input
- Prints Good Morning, Good Afternoon, Good Evening, or Good Night

## Concepts Used

- Variables
- User Input
- Type Casting
- Conditional Statements
- if-elif-else
- Basic Logic Building
